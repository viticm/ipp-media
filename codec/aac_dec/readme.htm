Overview

This sample illustrates how to use the Intel® Integrated Performance Primitives (Intel® IPP) signal processing and audio coding functions to implement a simple AAC (Advanced Audio Coding) decoder. The decoder supports the following features:

  1. Low Complexity Profile (AAC_LC) is implemented.
  2. Long Term Prediction Profile (AAC_LTP) is implemented
  3. Main Profile (AAC_MAIN) is implemented
  4. Scalable Sampling Rate Profile (AAC_SSR) is implemented
  5. MPEG-4 SBR (Spectral Band Replication) extension is implemented
  6. PS (Parametric stereo) is implemented
  7. IS, MS, TNS, PNS tools are supported
  8. ADIF, ADTS, MP4 formats are supported.

Additional information on this software as well as other Intel software performance products is available at http://developer.intel.com/software/products/ .

Known Limitations

The decoder supports the standard ISO/IEC 14496-3 with certain restrictions. Only the following profiles are supported in the current version:

  • Low Complexity Profile (AAC_LC)
  • Long Term Prediction Profile (AAC_LTP)
  • Main Profile (AAC_MAIN)
  • Scalable Sampling Rate Profile (AAC_SSR)
  • Spectral Band Replication (SBR)
  • Parametric Stereo (PS)
