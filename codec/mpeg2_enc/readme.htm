Overview

This sample demonstrates how to use the Intel ® Integrated Performance Primitives (Intel ® IPP) to create a MPEG-2 encoder. Currently MPEG2 encoder supports Simple, Main and High profiles without enhancement layers. 

Additional information on this software as well as other Intel software performance products is available at http://developer.intel.com/software/products/ .

Known Limitations

The encoder supports the standard ISO/IEC 13818-2 with certain restrictions in the current version:

  • Dual-prime prediction is not used in motion estimation
  • VBV control sometimes allows buffer overflow
  • Scalability is not supported
  • ISO/IEC 11172-2 (MPEG 1) output is not supported.
