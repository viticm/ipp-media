Overview

This sample illustrates how to use the Intel ® Integrated Performance Primitives (Intel ® IPP) color conversion functionality to implement UMC::ColorSpaceConverter component.

Additional information on this software as well as other Intel software performance products is available at http://developer.intel.com/software/products/ .
