Overview

This sample demonstrates how to use the Intel ® Integrated Performance Primitives (Intel ® IPP) to create an H.261 encoder. Currently H.261 encoder supports the ITU-T Recommendation H.261 (03/93) with the exception of switched multipoint operation, forward error correction, and still image transmission.

Additional information on this software as well as other Intel software performance products is available at http://developer.intel.com/software/products/ .

Known Limitations

The encoder supports the standard ITU-T Recommendation H.261 (03/93) with certain restrictions. Only the following features are supported in the current version:

  • I-Frames
  • P-Frames
  • Loop Filter
