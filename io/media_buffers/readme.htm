Overview

This folder contains implementations of  several classes, inherited from UMC::MediaBuffer interface. These classes are used to minimize memory copy and organize efficient data flow in media applications. Methods of UMC::MediaBuffer class provide asynchronous access to flow of data and allow using minimized portions of data for both components (producer & consumer) during data passing. Another benefit is that hiding data buffering code in MediaBuffer object simplifies the application and other components code.

Additional information on this software as well as other Intel software performance products is available at http://developer.intel.com/software/products/ .
