Overview

This folder contains implementations of file reading/writing operations, based on UMC::DataReader and UMC::DataWriter interfaces. This functionality is used in UMC based applications to read data from source file or to output it file

Additional information on this software as well as other Intel software performance products is available at http://developer.intel.com/software/products/ .
